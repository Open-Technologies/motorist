# Motorist

JavaScript game

## Used

* [ThreeJS](http://threejs.org/)
* [React](http://facebook.github.io/react/)
* [Babel](https://babeljs.io/)
* [Browserify](http://browserify.org/)

## Commands

### Installation

```sh
npm i
```

### Build and run

```sh
gulp
```

### Run

```sh
npm start
```

[http://localhost:8080/editor](http://localhost:8080/editor) - Map editor
